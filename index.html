AI 網路情蒐

賽前輸入對手隊名，AI 會搜尋公開網站（就是棒、賽事官網、新聞報導等）整理對手近況、需要留意的指標人物與應對說明。

AI 會同時搜尋聯盟成績網站與新聞報導；賽事欄留空則全網搜尋。網路上能查到的資訊依球隊而異，若查不到 AI 會如實回報，可改貼成績頁網址或手動建檔。
📋 半自動情蒐（此環境連不上網時使用）
流程：① 按下方按鈕複製提問 → ② 貼到 Claude 對話（App 或網頁版皆可）送出 → ③ 把 Claude 的完整回覆貼回下方 → ④ 解析存檔。上方的對象與隊名欄位共用。

指定成績網頁分析

已知對手在就是棒或其他網站的成績頁時，貼上網址，工具會讀取頁面內容交給 AI 整理成情蒐報告。

透過 allorigins.win 代理讀取，部分網站可能擋代理讀不到；失敗時改用上方網路情蒐或手動建立。

情蒐報告

比賽卡片會自動連結同名對手的情蒐報告，賽前打開就能複習。

匯入協作統計資料

從 Excel / Google 試算表整理好資料後，直接複製貼上（支援逗號或 Tab 分隔，第一列若是標題會自動略過）。找不到的球員與比賽會自動建立。

先選好上方「資料類型」與「階級」再匯出範本，會依該階級現有球員自動填好姓名欄，記錄員只要在 Excel／記事本填數字存檔，再用「選擇CSV檔匯入」讀回、確認內容後按「開始匯入」即可。

分享戰報

所有人都要用 Google 帳號登入並經你核准後才能檢視。三種身分：管理者可編輯並管理成員；編輯者可登錄／修改數據；唯讀成員只能瀏覽。新登入者預設為「待核准」，核准後才看得到資料。

AI 功能設定

  

    
AI 功能（網路情蒐、網頁分析、判定自責分、評選 MVP、個人分析、賽後焦點總結）需要 Anthropic API Key 才能運作，費用依實際用量從你的 Anthropic 帳戶扣。Key 只存在 Firestore、僅管理者與編輯者可讀取。功能僅開放管理者與編輯者；編輯者每個功能每天限用下方設定的次數，管理者不受限。

    

      

        

      載入中…
    

    

      

      

      
      
    

    
取得 Key：到 console.anthropic.com 註冊 → Billing 儲值（最低 5 美元）→ API Keys 建立一把貼到上方。建議同時在 Console 設定每月用量上限。「編輯者每日次數」填 0 = 不開放編輯者使用。


    
各模型價格（估算花費用）

    
Anthropic 沒有提供查詢帳戶餘額的 API，所以無法直接顯示 Console 那樣的即時餘額。這裡改成：依你填的價格，估算下方「AI 用量花費」實際打了多少 token、花了多少錢。價格請到 console.anthropic.com → Model pricing 核對後填入（USD / 每百萬 tokens），留空的模型不會估算金額。
